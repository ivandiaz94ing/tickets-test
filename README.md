# PruebaTecnica
1. clonar proyecto
2. ```docker-compose up -d```
3. Clonar en archivo ```.env.templete``` y renombrarlo a ```.env```
4. Cambiar las variables de entorno
5. Levantar la base de datos
```
docker-compose up -d
```
6. Levantar la aplicacion
```
npm run start:dev
```
